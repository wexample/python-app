# wexample-app

Version: 0.0.53

Helpers for building Python app or cli.

## Installation

```bash
pip install wexample-app
```

## Introduction

Provides essential base classes that facilitate quick development of simple, well-structured CLI programs.

## Quickstart

Get started with **wexample-app** (version 0.0.53) in seconds:

```python
# Your first example with wexample-app
from wexample-app import YourClass

# Initialize and use
instance = YourClass()
instance.run()
```

This package requires Python >=3.10.
