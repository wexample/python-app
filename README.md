# wexample-app

Helpers for building Python app or cli.

Version: 0.0.53

## Introduction
Provides essential base classes that facilitate quick development of simple, well-structured CLI programs.

## Requirements

- Python >=3.10

## Dependencies

- attrs>=23.1.0
- cattrs>=23.1.0
- dotenv
- pydantic>=2,<3
- python-dotenv
- wexample-filestate==0.0.54
- wexample-helpers-yaml==0.0.70

## Installation

```bash
pip install wexample-app
```
