# wexample-app

Helpers for building Python app or cli.

Version: 0.0.47

## Requirements

- Python >=3.10

## Dependencies

- dotenv
- pydantic>=2,<3
- python-dotenv
- wexample-filestate==0.0.47
- wexample-helpers-yaml==0.0.65

## Installation

```bash
pip install wexample-app
```
